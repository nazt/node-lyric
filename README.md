# INSTALLATION

`npm install -g lyric`

# USAGE
    Usage: /usr/local/bin/lyric [keyword] [options]

    Options:
      -i, --itunes  Get song name from itunes
      -c, --clean   Show only lyric          
      -s, --set     Set lyric of current song to itunes
      -t, --title   Show song title
